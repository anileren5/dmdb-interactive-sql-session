# DMDB - Interactive SQL Session 2026

## Prerequisites
Ensure you have the following installed:
- Docker
- Docker Compose

## Tutorials
This project includes tutorials in the `tutorials` folder.
- Data Definition Language (DDL)  
- Views  
- TPC-H Queries  

Each tutorial comes in two versions:
- A clean version for hands-on practice
- A reference version with all cells executed

## Getting Started
To launch the environment, run:
```bash
docker-compose up -d
```

Then open [http://localhost:8888](http://localhost:8888) in your browser to access the Jupyter server.

## Shut Down
To stop the environment:
```bash
docker-compose down
```
